# Testing Practice

JavaScript testing practice project built for The Odin Project. This project uses Jest to test JavaScript utility functions.

## Features

- Capitalize a string
- Reverse a string
- Calculator object
- Caesar cipher
- Analyze an array

## Built With

- JavaScript
- Jest
- Babel

## What I Learned

- How to write unit tests
- How to use Jest matchers
- How to follow TDD
- How to test edge cases
- How to organize JavaScript functions into modules

## Run Tests

```bash
npm test
```
